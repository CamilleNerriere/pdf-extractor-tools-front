Pdf tools

Pdf tools est un outil visant à faciliter le travail académique avec les pdfs. Deux outils sont proposés :

  • Un extracteur de citations permettant d'analyser un pdf et d'extraire les citations d'ouvrages trouvées (citations classiques entre guillemets, citations de type Harvard – (Smith, 2014, p. 33), et citations de type bloc). À chaque citation est associée sa référence.
  • Un extracteur d'annotations permettant d'extraire les annotations d'un pdf.

Trois types d'export sont proposés : .pdf, .txt et .word.

Note : il s'agit d'une version de démo, l'inscription est donc désactivée.
